#### 将文件移动到回收站

* 请求地址 /v1/files/recycle ```POST```
* 请求方式 JSON/Http POST
* 请求参数

| Properties     |  Type  | Description                                          |
|----------------|--------|------------------------------------------------------|
| * uuid         | string | 需要重命名该文件(夹)的uuid(可选) 可以为数组              |
| * path         | string | 需要重命名文件(夹)的路径(可选)，可以为数组                 |




#### 示例

* 将uuid为 561 的目录移动到回收站

发送 /v1/files/recycle :
```json
{
	"uuid":"561"
}
```
收到:
```
{
    "status": 200,
    "result": true,
    "code": "OK",
    "success": true,
    "token": "..."
}
```

*  使用path更改

发送 /v1/files/recycle :
```json
{
	"path":"/a/b/c/d"
}
```
收到:
```
{
    "status": 200,
    "result": true,
    "code": "OK",
    "success": true,
    "token": "..."
}
```
