## 以分页方式列出文件夹

* 路径: ```/v2/files/page```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| path          	| string  	|        	| 是   	| 文件夹路径           	|   单个路径不得超过128字符   	|
| page          	| long  	|        	| 是   	| 页码              	|   默认为1   	|
| pageSize         	| long  	|        	| 是   	| 单页显示的数量          	|   默认为20   	|



* 您也可以用 ```identity``` 获取路径

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| identity         	| string  	|        	| 是   	| 文件夹id            	|                     	|
| page          	| long  	|        	| 是   	| 页码              	|   单个路径不得超过128字符   	|
| pageSize         	| long  	|        	| 是   	| 单页显示的数量          	|   单个路径不得超过128字符   	|

#### 示范

* 发送

```json
{
	"path":"/files/papaya/create/f@乐器",
	"name":"木吉"
}
```
