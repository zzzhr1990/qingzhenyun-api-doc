## 添加离线任务

* 与 v1相比，一次请求即开启离线任务，更加简洁。

### 1、添加离线任务

* 路径: ```/v2/offline/add```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| url   	| string  	| 86     	| 默认 	| 路径，一行一个               	| 用户国家码                   	|
| path         	| string  	|        	| 是   	| 存储的用户本地路径       	|   用户存储路径，不接受identity                   |

* 用户存储路径，不接受identity，原因是因为离线下载为异步操作，此过程中可能删除文件夹

#### 示范

* 发送 

```POST``` ```/v2/offline/add```

```json
{
	"path": "/",
	"url": "http://www.baidu.com/index.html"
}
```

* 接受

```json
{
    "status": 0,
    "result": {
        "files": [],
        "success": true,
        "count": 1
    },
    "code": "OK",
    "success": true
}
```
